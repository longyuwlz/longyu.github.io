# 追光者
```
阳光，被绵延数里的云朵遮挡。
我顶着这被云朵拥抱着的太阳，
漫无目的地，漫无目的地，
骑车去往一片温暖的草场。

风在我耳旁呼啸张狂，
我正踩着踏板一圈圈前进着。
一个个红绿灯、一颗颗树木的身影匆匆掠过，
阳光渐渐钻破厚重的云朵。

这温暖的阳光撒向土坡，
在我的身上发光发热。
我就静静地坐在草地上，
在这不知名的，有着两三树木点缀的草地上。

这时，心中那潜藏着的声音又开始讲述。
讲出了内心中深切的渴望与想象。
想象着，她正从太阳高挂的地方向我招手，
向我轻轻地微笑。
她原来便是一束光！

她突然转身，向着落日的余晖奔跑，
我也追随着她的脚步。
我就这样跟在她的身后，
走过一个个路口。
抬头时她却已经过。
终于在彩霞满天时失去了她所有的光芒！
只剩下头顶的夜空。

夜空沉寂，我的眼里满含着泪水。
这一场追逐光芒的游戏有何结局？
原来光芒早已留在了我的心里！
```
